# Goldman Sachs - Business Analysis Report

A comprehensive McKinsey-style business analysis of Goldman Sachs Group, inc. (NYSE: GS), covering FY 2022-2024 financial data.

## What's inside
**PDF Report** - 14 section analyst report with branded cover, charts, and scorecard
**Excel Workbook** - 7 structured sheets with live formulas, colour-coded signals, and visuals

## Analysis Covers
- Financial Performance (Revenue, ROE, EPS, Margins)
- Business Model Canvas
- Market & Industry Analysis (TAM/SAM/SOM, Porter's Five Forces)
- Competitive Positioning (BCG Matrix, competitor benchmarking)
- SWOT + PESTLE Analysis
- Customer Segmentation & RFM Analysis
- Risk Analysis (Likelihood x Impact Matrix)
- Growth Strategy (Ansoff Matrix)
- Analyst Verdict: **Strong Buy / Strategic Partner**

** Tools used
- Python (ReportLab, OpenPyXL)
- McKinsey analytical frameworks
- Data sourced from GS public filings (10-K, Annual Report 2024)
